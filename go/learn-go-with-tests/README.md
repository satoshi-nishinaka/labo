## Goのテストコマンド

### 
```bash
$ go test
```

### サンプル関数も含めて実行
```bash
$ go test -v
```

### ベンチマーク計測
```bash
$  go test -bench=.
```

### カバレッジ計測
```bash
$ go test -cover
```
